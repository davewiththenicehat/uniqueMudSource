# Contributing source code
This project uses python. In an Evennia enviroment. To log into the server please refer back to the readme in the main document.

## Getting Started
Check if python is installed with `python --version`
You will need version 3.7 or later

If you do not have python
Install python from: https://www.python.org/downloads
Selecting your OS. You will need to download a version 3.7 or newer.
When done verify
